Trajectories and the Phase Plane

Recap: The Phase Plane

The phase plane consists of

  • Axes labeled with the state variables
  • Nullclines for each variable (preferrably in different colors)
  • Equilibria labeled by their coordinates.

We will add more to this, but when I ask you to “make a phase plane,” these are the essential three ingredients.

Example 1

Make a phase plane for the following system of differential equations. (Side note: I just made this example up out of thin air.) \[ \begin{align*} \frac{dx}{dt} &= y x - 2y^2\\ \frac{dy}{dt} &= 2x + 1 \end{align*} \]

First, let’s find the nullclines. For \(x\), we’ll have \(y(x-2y)=0\), giving two figures: the horizontal line \(y=0\) and the slanted line \(y=\frac{1}{2}x\).

For the \(y\) nullclines, we only get a vertical line \(2x+1= 0\), or \(x=-\frac{1}{2}\). Here’s what it looks like. Notice we only have two equilibria.

What are those equilibria? The \(x\) coordinates are easy: both are \(-0.5\). For the top equilibrium, you get \(y=0\), so that equilibrium is \((x^*,y^*)= (-0.5, 0)\). For the other, it lies on the line \(y=0.5 x\). Therefore, the \(y\)-value is \(0.5(-0.5) = -0.25\). We get the equilibria \[ (x^*,y^*) = (-0.5, 0) \quad \text{ and } \quad (x^*,y^*) = (-0.5, -0.25). \]

Trajectories

Remember, the whole point of a system of differential equations (or any differential equation whatsoever) is to know what the solutions \(x(t)\) and \(y(t)\) do in time. The way we show this on the phase plane is through a trajectory.

Definition. A trajectory is a path in the phase plane that shows the values of \(x(t)\) and \(y(t)\) in time.

Let’s understand first what a trajectory looks like and what it tells us.

Direction Fields, or how trajectories are made

We get trajectories from the system of differential equations. We can either ask a computer to make a very accurate trajectory, or we can sketch them by hand. Computers make them with a two-dimensional version of Euler’s Method. We do have a way of making sense of them by hand, at least approximately. We use what’s called a direction field.

Here’s what you do: pick a point \((x,y)\) in the phase plane. Then plug these into the system to get \((dx/dt, dy/dt)\) values (like you were about to do Euler’s method). This gives you two different numbers, and then use these to make an arrow.

Example 5

Let’s return to the system we had before: \[ \begin{align*} \frac{dx}{dt} &= y x - 2y^2\\ \frac{dy}{dt} &= 2x + 1 \end{align*} \] Let’s pick the point \((x,y)=(1,1)\). Doing so gives you \(\frac{dx}{dt} = -1\) and \(\frac{dy}{dt} = 3\), producing the pair \((-1,3)\). This is an arrow pointing to the left by 1 and up 3. [Side bar: if you know vectors, this is a vector.]

If we do \((x,y)=(1,0)\), we get \(\frac{dx}{dt} = 0\) and \(\frac{dy}{dt} = 3\). We would draw this as a vector pointing upwards.

Summary of Phase Planes

Here’s the summary for all of the components of a phase plane. Given any system of two differential equations, we perform the following:

  1. Make a set of axes with the state variables.
  2. Make nullclines and identify equilibria.
  3. Make a bunch of direction vectors using the differential equations.
  4. Make a few trajectories that follow the direction vectors.
  5. Turn trajectories into information about the state variables \(x\) and \(y\).
